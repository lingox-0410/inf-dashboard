🎯 本 Tab 回答的 3 个业务问题

Q1 课时下降的根因？
教练数稳定 → 找人效降的根因；教练数缩水 → 招聘排班。
Q2 是会员少 还是 上课少？
会员稳定 → 提高密度（21日打卡）；会员减少 → 拉新（体验课）。
Q3 收入下滑：量 还是 价？
价跌 → 大单户挽回；量跌 → 拉新+续费同步。
三个等价公式： 课时 = 教练数 × 人均教练课时 ｜ 课时 = 活跃会员 × 月均频率 ｜ 收入 = 课时 × 课时单价

📐 附录 · 全局指标口径与计算规则

所有 Tab 共用

📚 课时类指标

  • • 课时数 = 私教课时统计中"上课节数"加总（2025-05 至 2026-05）
  • • 当月课时 = 历史课时记录 + 排课记录已完成(增量补齐)
  • • 人均课时 = 课时数 ÷ 该期内有上课的会员数
  • • 月均次数 = 会员累计课时 ÷ 跨度月数
  • • 月均频率（L3公式2）= 月度总课时 ÷ 当月活跃会员数
  • • 人均教练课时（L3公式1）= 月度总课时 ÷ 当月有上课的教练数

👥 会员类指标

  • • 会员去重 = 姓名 + 手机号
  • • 服务会员/上课会员 = 当月有上课的会员数（不重复）
  • • 主带会员 = 该教练为某会员上课最多 → 该会员归为该教练
  • • 活跃 / 稳定 / 冷却 / 风险 / 流失 = 距今天数 ≤14 / 30 / 60 / 120 / >120
  • • 在练私教会员 = 总剩余课时 > 0 的会员

🏃 教练类指标

  • • 总课时 = 该教练上课记录加总（按时间筛选区间）
  • • 人效 = 课时数 ÷ 该期服务会员数
  • • 维护率 = (活跃+稳定 主带会员) ÷ 主带会员数
  • • 流失率 = 流失会员 ÷ 主带会员数

📚 课程 + 💰 营收

  • • 课程归类 = 按课程名称语义聚合（增肌/减脂/普拉提/纠正/孕产/拳击...）
  • • 关联营收 = 该归类下会员的购买金额加总（按 会员+课程名+金额 去重）
  • • 客单价 = 关联营收 ÷ 会员数
  • • 课时单价（L3公式3）= 月度收入 ÷ 月度课时数
  • • 复购率 = 同归类下购买>1次的会员 ÷ 总会员
  • • 月度收入 = 收银流水中 金额>0 当月的加总（剔除退款）

💎 剩余课时（核心规则）

  • • 课时包剩余 = 正式私教卡的剩余购买 + 剩余赠送（按次）
  • • 储值卡剩余 = 储值-其他卡 (剩余购买+赠送 元) ÷ 498
  • • 总剩余 = 课时包剩余 + 储值卡剩余 （同一会员合并）
  • • 剔除：体验课/裂变券/赠课/团课/团课储值/活动券
  • • 状态过滤：仅算「正常 + 请假中」状态的卡

🔥 续课/跟进标签

  • • 🔥 紧急续课 = 总剩余 ≤ 5 节（且 >0）
  • • 💰 需续课 = 5 < 总剩余 < 20 节
  • • ⏰ 久未来 = 总剩余 ≥ 2 节 且 距今 ≥21 天
  • • 📉 频率下降 = 近3月均 ≥3 节 且 当月课时 < 近3月均 × 0.5

🏷 L4 五个复杂标签

  • • ① 流失预警 0-100 = 频率衰减(25) + 距今(25) + 剩余少(25) + 主教练独占(25)
  • • ② 教练连带 = 主教练课时占比 >80% + 剩余≥10 → 🔴 高危独占
  • • ④ NBA续费分 0-100 = 消费(25) + 课时(25) + 近30天(25) + 紧迫度(25)
  • • ⑥ 九宫格 = 累计消费三分位 × 月均次数三分位
  • • ⑩ 转介绍贡献 = 客户来源="转介绍" + 累计消费≥1万

🎟 体验课 (L6)

  • • 体验课成交率 = "已成交" ÷ 总数
  • • 含高潜成交率 = ("已成交" + "高潜成交") ÷ 总数
  • • 客单价（成交） = 成交金额 ÷ 已成交人数
  • • 未成交原因 = 按语义聚合 8 类（价格/距离/考虑/暂不需要/已有/健康/失联/其他）
